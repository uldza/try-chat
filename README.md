# TRY-CHAT #

## Requirements ##

* Puma web server
* Redis


## Technologies used

* reactJS
* websockets
* altJS (for flux implementation)
* es6
* Bootstrap
* ruby on rails


## Test run instructions ##

* to run tests `rspec`
* to run tests with coverage report `COVERAGE=y rspec`

## PS ##
To create admin user and guest user, run `rake db:seed`
